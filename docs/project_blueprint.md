# PersonaMood-AI

## Overview

PersonaMood-AI is an intelligent conversational assistant that personalizes responses using:

- Personality Detection
- Mood Detection
- Prompt Engineering
- GPT
- Conversation Memory

---

## Objectives

- Detect user's Big Five personality.
- Detect current emotional state.
- Adapt prompts dynamically.
- Generate personalized responses.
- Maintain conversation context.
